## Deploy — GitHub + Netlify

### Passo 1: Subir para o GitHub
```bash
# Dentro da pasta venda/
git init
git add .
git commit -m "feat: página de vendas Wasender 5.0.8"
git branch -M main
git remote add origin https://github.com/SEU_USUARIO/wasender-vendas.git
git push -u origin main
```

### Passo 2: Conectar Netlify ao GitHub
1. Acessa netlify.com → "Add new site" → "Import an existing project"
2. Conecta com GitHub e seleciona o repositório `wasender-vendas`
3. Build command: deixar vazio
4. Publish directory: `.` (raiz)
5. Clica "Deploy site"

### Passo 3: Atualizações futuras
Qualquer alteração no HTML basta fazer:
```bash
git add .
git commit -m "fix: descrição da alteração"
git push
```
O Netlify detecta automaticamente e republica em segundos.

## Manutenção Comum

### Trocar link de compra (Hotmart)
Buscar no index.html por `href="#"` dentro da seção `#planos` e substituir pela URL da Hotmart.

### Adicionar novo vídeo tutorial
Adicionar novo `.cslide` no carousel com o ID do YouTube:
```html
<div class="cslide">
  <div class="tcard" data-video="ID_YOUTUBE">
    <img src="https://img.youtube.com/vi/ID_YOUTUBE/hqdefault.jpg" alt="Título">
    <div class="tplay"><div class="tplay-btn">▶</div></div>
    <div class="tlabel">Aula XX — Título</div>
  </div>
</div>
```

### Alterar preços
Buscar `.pprice` e `.pper` nos dois `.pcard` dentro de `#planos`.

### Adicionar depoimento
Copiar um bloco `.tcard2` dentro de `.tgrid` e alterar texto, inicial do avatar e nome.

## Observações Técnicas
- Vídeos abrem em modal (iframe lazy) — só funcionam via http:// ou https://, não via file://
- Para testar localmente: `python3 -m http.server 8080` e acessar `http://localhost:8080`
- Todas as imagens de thumbnail dos vídeos são carregadas do YouTube (externas)
- As duas imagens locais (`logowasender.png` e `versao-5.0.1.png`) devem estar na mesma pasta que o `index.html`
